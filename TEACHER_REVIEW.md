## 一、总体评价

## 二、内容准确性问题

### 2.1 化学方程式相关

| 实验 | 问题 | 建议修改 |
|------|------|----------|
| 实验二（exp-02） | 方程式 `2KMnO4 -> K2MnO4 + MnO2 + O2` 缺少反应条件标注"加热（△）"和气体符号"O2↑" | 改为 `2KMnO4 --△--> K2MnO4 + MnO2 + O2↑` |
| 实验三（exp-03） | 方程式 `2H2O2 -> 2H2O + O2 (MnO2)` 催化剂应标在箭头上方而非括号 | 改为 `2H2O2 --MnO2--> 2H2O + O2↑`，且应标注 O2↑ |
| 实验一（exp-01） | 方程式 `4P + 5O2 -> 2P2O5` 缺少"点燃"条件标注 | 改为 `4P + 5O2 --点燃--> 2P2O5` |
| 实验五（exp-05） | 方程式 `2H2O -> 2H2 + O2` 缺少"通电"条件和气体符号 | 改为 `2H2O --通电--> 2H2↑ + O2↑` |
| 实验十（exp-10） | 方程式 `2CuO + C -> 2Cu + CO2` 缺少"高温"条件和气体符号 | 改为 `2CuO + C --高温--> 2Cu + CO2↑` |
| 实验十一（exp-11） | 两个方程式均缺少条件/符号标注 | `CO2 + H2O <-> H2CO3` 可保留；`CO2 + Ca(OH)2 -> CaCO3↓ + H2O` 应确保↓符号 |
| 实验二十（exp-20） | 方程式 `Fe2O3 + 3CO -> 2Fe + 3CO2` 缺少"高温"条件 | 改为 `Fe2O3 + 3CO --高温--> 2Fe + 3CO2` |
| 实验四（exp-04） | 方程式写的是 `NH3 + H2O <-> NH3.H2O`，应使用化学规范写法 | 应写为 `NH3 + H2O = NH3*H2O`（初中不要求可逆符号，用"="即可；或保留可逆但需说明） |
| 实验十九（exp-19） | 方程式 `HCl + NaOH -> NaCl + H2O` 正确，但中考中一般要求写"=" | 全站统一：用"="还是"->"应保持一致。中考答题要求用"=" |

**核心建议**：中考评分对方程式要求极为严格，**缺少反应条件（点燃/加热/高温/催化剂/通电）和生成符号（↑↓）直接扣分**。这是本站最需要优先修复的问题。建议在 `exp_notes.json` 中将所有方程式补全条件和符号，并在 HTML 渲染时正确显示。

### 2.3 具体内容错误或不严谨

1. **实验一（exp-01）**：考点中"结果偏大常见原因"提到"止水夹未夹紧"，这里原因解释不够明确。止水夹未夹紧应该是导致偏小（空气进入），而非偏大。**偏大的原因是：点燃红磷伸入时动作慢，导致瓶内空气受热膨胀从导管逸出，冷却后外界大气压推动更多水进入**。建议修正表述。

2. **实验三（exp-03）**：步骤中写"收集完毕先移出导管，再停止反应（防倒吸）"。对于 H2O2 分解实验（固液不加热型），控制反应速率是通过分液漏斗控制滴液量，不存在典型的"倒吸"问题。这里的表述应改为："停止滴加 H2O2 即可停止反应"。

3. **实验十（exp-10）**：安全提醒中"通气导管不要堵塞，防止倒吸"——措辞应更精确为"实验结束先撤导管后灭灯，防止倒吸；冷却过程中夹紧橡皮管，防止生成的 Cu 被重新氧化"。

4. **实验四（exp-04）**：方程式 `NH3.H2O` 应写为 `NH3*H2O`（中间是居中点而非英文句号），或直接写汉字名"一水合氨"。

5. **实验十五（exp-15）**：铁锈的化学式在 PDF 摘录中写为 `Fe2O3*xH2O`，但主内容中只说"含水的氧化铁"，建议在原理中补充化学式。

6. **实验十六（exp-16）**：标题用的是"配置"，正确的化学术语应为"配制"。这个错误出现在多个页面和 JSON 中。

## 三、考点覆盖度分析

### 3.1 已覆盖的核心考点（良好）

- 空气中氧气含量测定（必考）
- 两种制氧方法（KMnO4 加热 / H2O2 催化分解）
- 电解水实验（必考）
- CO2 的性质和制取（必考）
- 燃烧条件探究（必考）
- 金属活动性顺序（必考）
- 粗盐提纯（必考实验操作题）
- 配制一定质量分数溶液（必考实验操作题）
- 过滤/蒸馏/净水器（水的净化专题）
- 质量守恒定律验证

### 3.2 缺失或薄弱的重要考点

以下是广东中考（含深圳）化学实验部分经常考查但本站未充分覆盖的内容：

1. **氧气的化学性质实验**（重要！）
   - 铁丝在氧气中燃烧：`3Fe + 2O2 --点燃--> Fe3O4`（火星四射、生成黑色固体）
   - 硫在氧气中燃烧：`S + O2 --点燃--> SO2`（蓝紫色火焰）
   - 木炭在氧气中燃烧：`C + O2 --点燃--> CO2`
   - 镁在空气中燃烧：`2Mg + O2 --点燃--> 2MgO`（耀眼白光）
   - 这些是中考高频考点（"描述实验现象"题型），**建议新增一个实验页专门总结"氧气的化学性质"**

2. **溶解度曲线的理解与应用**
   - 虽然不是单独实验，但与实验十六、十七密切关联
   - 建议在相关实验中补充溶解度曲线读图要点

3. **pH 的测定方法**
   - 实验十九（酸碱中和）中提到了 pH 变化曲线，但缺少 **pH 试纸的使用方法**（中考必考操作）
   - 建议补充：取 pH 试纸放玻璃片上 -> 用玻璃棒蘸取待测液 -> 与标准比色卡对比

4. **气体的干燥方法**
   - 酸性干燥剂（浓硫酸）、碱性干燥剂（碱石灰）、中性干燥剂（CaCl2）
   - 这是广东中考实验题中的常见考点，建议在"实验基础知识"模块中补充

5. **实验方案设计题常用的对照实验思想**
   - 虽然实验十三、十五涉及了对照实验，但缺少系统总结
   - 建议新增一个"实验设计方法论"板块，涵盖控制变量法、对照实验设计原则

### 3.3 化学方程式配平练习

中考中化学方程式书写占很大分值，但本站只列出了方程式，缺少配平练习环节。建议新增交互式配平练习功能。

## 四、易错点标注评估

### 4.1 已标注的易错点（较好）

- 实验一的偏大/偏小原因分析
- 实验二的操作顺序"检装固点收移灭"
- 实验十二不用浓盐酸/稀硫酸的原因
- 实验十六的偏大/偏小分析

### 4.2 需要补充的高频易错点

| 实验 | 需补充的易错点 |
|------|----------------|
| 实验二 | "药品平铺"——不是堆放，是为了受热均匀；试管口向下倾斜的角度和原因 |
| 实验三 | 长颈漏斗下端必须伸入液面以下（形成液封防止气体逸出）——这是中考常考的装置改进题 |
| 实验五 | "正氧负氢"口诀要在考点中突出显示；水中加 NaOH/稀 H2SO4 的作用要明确（增强导电性，不能加 NaCl 因为会电解 NaCl） |
| 实验六 | "一贴二低三靠"中"三靠"的第三靠是"漏斗下端尖嘴靠烧杯内壁"——作用是引流，防止液体飞溅。当前描述不够完整 |
| 实验九 | 应补充：哪些实验不能在敞口容器中验证质量守恒（生成气体的反应如 Na2CO3+HCl）；哪些可以（沉淀反应如 NaOH+CuSO4） |
| 实验十一 | 石蕊纸花实验是三组对比的经典设计，要强调"为什么要三组"（排除单一因素）——中考常考 |
| 实验十三 | 白磷着火点约 40 度 C，红磷着火点约 240 度 C（注意：PDF 中写 260 度 C 存在争议，人教版教材写的是 240 度 C）——建议按教材标准 |
| 实验十七 | 玻璃棒在三个步骤中的不同作用（溶解时搅拌加速溶解、过滤时引流、蒸发时防暴沸）——这是中考简答题高频考点 |
| 实验十八 | 除杂顺序中 "BaCl2 必须在 Na2CO3 之前" 的原因要更清楚地解释 |
| 实验十九 | 中和反应中为什么选酚酞而不选石蕊——酚酞变色点接近中性（pH 8.2-10），更易判断终点 |
| 实验二十 | "先通 CO 再点燃"和"先停止加热后停通 CO"的原因——CO 是可燃气体，与空气混合可能爆炸 |

## 五、教学设计改进建议

### 5.1 内容呈现优化

1. **方程式的条件和符号必须可视化**
   - 当前在 HTML 中方程式的条件标注不明显，建议用专门的 CSS 样式在箭头上方显示反应条件（如"△""点燃""催化剂""高温""通电"），在产物后显示"↑""↓"符号
   - 这是最关键的改进——直接影响学生答题时的规范性

2. **增加"实验现象"独立板块**
   - 很多实验页面缺少明确的"现象"板块（phenomena）
   - 中考实验题的标准答题格式是："实验操作 + 实验现象 + 实验结论"，缺少现象会让学生在答题时丢分

3. **增加"典型例题"或"真题链接"板块**
   - 每个实验可以附 1-2 道广东/深圳中考真题（或改编题），让学生在复习后立即练习
   - 不需要在线作答，列出题目和参考答案即可

4. **自测清单应更具体**
   - 当前所有实验页面的自测清单几乎一模一样（通用模板），缺乏针对性
   - 建议为每个实验定制 3-5 个具体问题，例如：
     - 实验一："红磷能否用铁丝替代？为什么？"
     - 实验二："如果先灭火后移导管会怎样？"
     - 实验十二："为什么不用 Na2CO3 替代 CaCO3？"

### 5.2 建议新增的学习功能

1. **记忆口诀汇总页**
   - "检装固点收移灭"（制 O2 操作顺序）
   - "一贴二低三靠"（过滤）
   - "正氧负氢"（电解水）
   - "先通气后加热，先停止加热后停通气"（CO 还原实验操作）
   - 建议做成一个独立页面或首页卡片

2. **实验对比表格**
   - O2 的两种制法对比（KMnO4 vs H2O2）：药品、装置、条件、收集方法
   - O2 和 CO2 实验室制法对比：药品、装置、收集、检验、验满
   - 气体收集方法选择总结表
   - 粗盐提纯（难溶性杂质 vs 可溶性杂质）对比

3. **常见气体的检验/验满/干燥/除杂总结表**
   - O2：带火星木条复燃
   - CO2：澄清石灰水变浑浊
   - H2：点燃，淡蓝色火焰
   - CO：点燃，蓝色火焰（有毒！必须尾气处理）
   - 水蒸气：无水 CuSO4 变蓝
   - 这是中考填空题和实验题的常见考点

4. **实验仪器用途速查表**
   - 首页"实验基础知识"板块中提到了仪器分类，但只是文字概述
   - 建议增加关键仪器的图片 + 用途 + 注意事项（尤其是：量筒读数平视凹液面最低处、天平左物右码、酒精灯用外焰加热等）

### 5.3 学习路径优化

当前 20 个实验按教材章节线性排列，建议提供**按考点主题分类**的另一种浏览方式：

- **气体制取与性质专题**：实验 1、2、3、11、12、20
- **水与溶液专题**：实验 5、6、7、8、16、17、18
- **化学反应原理专题**：实验 9、10、13、19
- **金属专题**：实验 14、15、20
- **微观粒子专题**：实验 4、5

## 六、针对深圳/广东中考的特别提醒

### 6.1 广东中考化学实验题特点

1. **实验操作考试（满分 8 分）**：深圳中考有独立的实验操作考试，通常考查配制溶液、粗盐提纯、制取 O2/CO2 等操作。本站的实验 2/3/12/16/17 直接对应考试内容，但 **缺少操作规范的详细描述**（如"倾倒液体时标签朝手心""瓶塞倒放"等细节）。

2. **综合实验探究题（分值 10-12 分）**：广东中考最后一道大题通常是实验探究，要求学生：
   - 设计实验方案
   - 预测实验现象
   - 分析实验数据
   - 得出结论
   - 本站可以在每个实验页增加"探究延伸"板块

3. **信息给予题**：近年广东中考常出现给新信息让学生应用已知实验原理解决新问题的题型，建议在拓展实验中增加此类引导。

### 6.2 建议补充的深圳/广东高频考点

1. **铁丝在 O2 中燃烧**——"火星四射、生成黑色固体"（几乎每年必考的现象描述题）
   - 集气瓶底要放少量水或铺薄层细沙——防止高温熔融物溅落炸裂瓶底
2. **天平的使用规范**——左物右码、游码归零、称量纸（腐蚀性药品用玻璃器皿）
3. **量筒使用**——选择合适量程（略大于量取体积）、读数方法（平视凹液面最低处）
4. **酒精灯使用规范**——用外焰加热、不可倾倒添加酒精、灯帽盖灭不可吹灭、酒精着火用湿布盖灭

## 八、具体修改清单（供开发者参照）

### 8.1 `exp_notes.json` 需要修改的方程式

```
实验 1: "4P + 5O2 -> 2P2O5"         => "4P + 5O₂ ==点燃== 2P₂O₅"
实验 2: "2KMnO4 -> K2MnO4 + MnO2 + O2" => "2KMnO₄ ==△== K₂MnO₄ + MnO₂ + O₂↑"
实验 3: "2H2O2 -> 2H2O + O2 (MnO2)"    => "2H₂O₂ ==MnO₂== 2H₂O + O₂↑"
实验 5: "2H2O -> 2H2 + O2"              => "2H₂O ==通电== 2H₂↑ + O₂↑"
实验 10: "2CuO + C -> 2Cu + CO2"         => "2CuO + C ==高温== 2Cu + CO₂↑"
实验 11: "CO2 + Ca(OH)2 -> CaCO3↓ + H2O" (确保↓符号存在)
```
